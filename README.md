## 描述

* 支持单/多html文件开发,支持输出编译合并inline类型html、引入类型html、 umd模式js脚本
* 支持less, es6

## 开始
```bash
$ npm run dev
```


- 打包引入类型html
```bash
$ npm run build
```

- 打包inline类型html
```bash
$ npm run build
$ npm run inline-tpl
```
- umd模式js脚本
```bash
$ npm run build-js
```

舒适开发： chrome浏览器安装liveReload,开始你的装逼模式~~
